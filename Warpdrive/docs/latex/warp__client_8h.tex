\doxysubsection{Detailed Description}
Delivers functions for connecting and requesting data from a warp\+Server. 

The warp\+Client delivers functions for connecting and requesting data from a warp\+Server. The workflow starts by connecting to the server via the function \mbox{\hyperlink{warp__client_8h_a39f7c09aa4d511bb248871e1694ba74f}{warpclient\+\_\+query\+Server()}}, which sends a query to the server. Subsequently, calls to \mbox{\hyperlink{warp__client_8h_a781cba21127237f9bdd29e4ef67e8296}{warpclient\+\_\+get\+Data()}} are filling buffers until a NULL is returned, which indicates the end of the data transfer. Lastly, a call to \mbox{\hyperlink{warp__client_8h_ad8b41c650bf538d890acaf2c8e065458}{warpclient\+\_\+cleanup()}} releases all allocated buffers and structs.
